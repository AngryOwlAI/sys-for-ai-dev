# Portable Source-First Memory Example

## Scenario

A generated target system has a memory catalog and source registry.

## Generic Steps

1. Run `<TARGET_MEMORY_STATUS_COMMAND>`.
2. Search for the target concept or work packet.
3. Inspect source paths and registry rows returned by useful hits.
4. Record a preflight receipt when the memory evidence affects routing or claims.
5. Treat generated derivatives as reader surfaces unless a promotion decision exists.

## Expected Result

The operator can find relevant authority without treating memory as authority.
