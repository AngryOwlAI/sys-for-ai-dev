# Assurance Case Builder

Use this skill when a safety, privacy, security, compliance, or reliability claim needs an auditable argument.

## Procedure

1. State the claim in precise, testable language.
2. Identify assumptions, hazards, controls, and required evidence.
3. Link evidence paths to the exact claim they support.
4. Mark unsupported or partially supported claims as gaps.
5. Record residual risk and validation follow-up.

## Boundary

An assurance case organizes evidence. It does not increase the strength of evidence that is absent, weak, or indirect.
