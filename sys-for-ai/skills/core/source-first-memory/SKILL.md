# Generic Source-First Memory

A target system generated by sys-for-ai may adapt this skill to use memory as a navigation layer to registered source authority.

Memory lookup finds candidate evidence. Source files, registry rows, contracts, and approved decisions decide.

## Generic Procedure

1. Run the target system's memory status command.
2. Search or lookup relevant memory objects.
3. Inspect canonical source paths or registry rows named by useful hits.
4. Record preflight evidence when memory affects routing, requirements, work boundaries, handoffs, permissions, or claims.
5. Label generated derivatives and caches as noncanonical unless a promotion route explicitly says otherwise.
