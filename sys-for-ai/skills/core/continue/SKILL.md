# Generic /continue

A target system generated by sys-for-ai may adapt this skill to resume from tracked state, inspect the latest handoff, run memory preflight, select at most one bounded AgentJob, validate completion, and create a new handoff.

This template is not itself active development-system authority unless promoted through the local system's skill registry and runtime policy.

## Generic Procedure

1. Read the target system's continuation policy.
2. Load tracked program state.
3. Inspect the latest handoff if one exists.
4. Run memory preflight when memory can affect routing.
5. Select or reuse at most one authorized work packet.
6. Execute only inside that packet's boundary.
7. Validate completion evidence.
8. Write the next receipt or handoff before reporting completion.
